# V-Racer
Retro drift racing game created using [mgfw - Rust Mini Game Framework](https://github.com/Syn-Nine/mgfw).

### Controls:
- Throttle - Up Arrow
- Steering - Left/Right Arrows
- Confirm - Space
- Quit Race - ESC
